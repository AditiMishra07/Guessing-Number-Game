# 🎯 Number Guessing Game (Java)

## 📌 About
This project is a simple Number Guessing Game developed in Java. The program generates a random number, and the player has to guess it. The game provides hints until the correct number is guessed.

## ✨ Features
- Random number generation
- Too High / Too Low hints
- 5 attempts
- Simple console-based interface

## 🛠️ Technologies Used
- Java
- VS Code

## 📚 Concepts Used
- Variables
- Loops
- If-Else Statements
- Scanner Class
- Random Class

## ▶️ How to Run
1. Open the project in VS Code.
2. Compile the Java file.
3. Run the program.
4. Start guessing the number!
